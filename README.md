# Bhagesh's Birthday Website

A text-based adventure game for Bhagesh's birthday.

## How to Run Locally
1. Open `index.html` in your web browser.
2. That's it! No server required.

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub.
2. Push these files to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Source**, select `main` branch and `/ (root)` folder.
5. Click **Save**.
6. Wait a few minutes, and your site will be live!

## Features
- **Mobile-First Design**: Optimized for phones.
- **Audio**: Plays background music.
- **Game**: A Zork-style text adventure.
- **No AI**: Runs on simple Javascript logic (no TensorFlow).
